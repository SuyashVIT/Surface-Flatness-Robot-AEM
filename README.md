 # Surface Flatness Measurement System

This project presents a low-cost embedded system to measure surface flatness
using an ultrasonic sensor and MPU6050 IMU.

## Features
- Real-time Flatness Index (FI) calculation
- Stable readings on flat surfaces
- Detection of bumps and uneven regions
- Live data logging and graph visualization

## Hardware Used
- Arduino UNO (ATmega328P)
- Ultrasonic Sensor (HC-SR04)
- MPU6050 IMU
- 16x2 I2C LCD

## Software Used
- Arduino IDE
- Python (Matplotlib, PySerial)

## How to Run
1. Upload Arduino code to UNO
2. Connect Arduino via USB
3. Run Python script to log data
